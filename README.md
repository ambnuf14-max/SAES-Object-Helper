# SAES Object Helper

**Инструмент расстановки объектов для SA:MP (MoonLoader)** — живое 3D-превью, удобный редактор и установка объектов прямо на сервер.

## Возможности

- **Палитра объектов** — ячейки быстрого доступа с **живым вращающимся 3D-превью** модели (рендер через EntityRender); сетка 2×2…5×5, смена модели на лету, добавление по ID.
- **Полупрозрачные превью** — объекты ставятся «призраками»: видно, как они впишутся в окружение, ещё до установки.
- **Режим редактирования (Edit)** — перемещение и вращение **по осям** (активная подсвечивается прямо в мире), **прижатие к поверхности** с выбором грани, привязка к сетке, сдвиги камеро-относительно.
- **Инструмент «Лента»** — целый инструмент для ограждений / полицейской ленты: сплайн или прямые, прогиб, глубина утопления, стыки, правка точек (полный разбор — в разделе «Режим Лента» ниже).
- **Эффекты и частицы** — поддержка моделей-эффектов (для них в ячейке показывается иконка вместо меша).
- **Подписи #N** — 3D-текст над каждым превью, включается/выключается клавишей.
- **Серверные объекты** — установка превью **прямо на сервер** (эмуляция RPC), по одному; отдельное меню со списком установленных и удалением по радиусу или поштучно.
- **Пресеты** — сохранение и загрузка **именованных наборов** объектов.
- **Undo / Redo** — полная история действий (`Ctrl+Z` / `Ctrl+Y`), включая **откат установки на сервер** (удаляет объекты с сервера и возвращает превью на места).
- **Гибкие хоткеи** — любую клавишу можно переназначить **прямо в меню** (с захватом и подтверждением).
- **Авто-обновление** — скрипт сам подтягивает новые версии из GitHub.

## Как это работает

Полный путь — от пустого места до готовой сцены **прямо на сервере**. Любой шаг отменяется (`Ctrl+Z`), так что экспериментировать безопасно.

### Шаг 1 · Открыл меню и выбрал объект
Команда `/objhelper` (или клавиша `O`) открывает меню. Сверху — **палитра**: ячейки быстрого доступа, и в каждой крутится **живая 3D-модель** объекта (рендер через EntityRender), так что видно, что берёшь, ещё до установки.
- Сетку ячеек можно сделать плотнее или крупнее — `2×2` … `5×5`.
- Новый объект добавляется **по ID модели** в один клик.
- Цифрами `1`–`9` выбираешь активную ячейку — это объект «в руке».
- Для моделей-**эффектов** (частицы и т.п.), которые нельзя показать мешем, в ячейке рисуется иконка.

<div align="center"><img src="assets/palette.png" width="55%" alt="Палитра с живым превью"></div>

### Шаг 2 · Взял объект «в руку»
`Alt`+цифра — и выбранный объект появляется перед тобой **полупрозрачным «призраком»**, который следует за прицелом. Полупрозрачность не для красоты: сразу видно, как объект ляжет в окружение, ещё до фиксации.
- `Alt` (зажать) — курсор-режим: крутишь камеру и целишься, не закрывая меню.

<!-- screenshot: ghost-объект перед игроком (полупрозрачный, следует за прицелом) -->

### Шаг 3 · Посадил точно — snap и сетка
- **Прижатие к поверхности** (`Alt+Shift`) — объект прилипает к земле, стене, бордюру или другому объекту, без зазоров и провалов. `R` циклит **грань прилегания**, если автоматика выбрала не ту.
- **Привязка к сетке** — для ровных рядов и симметрии.
- **Сдвиги камеро-относительно** — двигаешь объект «от себя / к себе / влево-вправо» относительно взгляда, а не мировых осей.
- Поставить — `Enter` (или `Alt+ЛКМ`).

<!-- screenshot: ghost, прижатый к поверхности (виден маркер грани прилегания) -->

### Шаг 4 · Повернул как надо — вращение по осям
Вращение — колесом мыши, и **активная ось подсвечивается прямо в мире** (на скрине — красная), так что крутишь именно вокруг нужной.
- `X` — сменить ось, `CapsLock` — сменить плоскость.
- Шаг поворота: обычно `15°`, `Shift` — `45°`, `Ctrl` — `1°` для тонкой подгонки.
- `V` — показать/скрыть оси, `R` / `Shift+R` — сбросить поворот.

<div align="center"><img src="assets/edit-axis.png" width="60%" alt="Активная ось при вращении"></div>

### Шаг 5 · Расставил несколько — и не запутался
Объектов может быть много. У каждого превью — **3D-подпись `#N`** над ним (вкл/выкл клавишей `N`), а стрелками `←`/`→` переключаешься между ними; активный подсвечивается.

<!-- screenshot: несколько превью с подписями #1, #2, #3 в мире -->

### Шаг 6 · Построил ограждение «Лентой»
Нужна полицейская лента или забор — включаешь **«Ленту»**: ставишь опорные точки, между ними сама тянется лента плавным сплайном (или прямыми — `M`). Snap работает и тут — лента ложится ровно по бордюру.

<div align="center"><img src="assets/tape-snap.png" width="55%" alt="Опорная точка ленты и snap"></div>

Лента — полноценный инструмент: прогиб, глубина утопления, стыки, правка отдельных точек. Полный разбор — в разделе **«Режим Лента»** ниже.

<img src="assets/tape-edit.png" width="49%" alt="Лента между точками"> <img src="assets/tape-sag.png" width="49%" alt="Вертикальный прогиб ленты">

### Шаг 7 · Ошибся — откатил
Любое действие отменяется `Ctrl+Z` и возвращается `Ctrl+Y` — это **полная история**, а не только последний шаг. Можно смело пробовать и переделывать.

### Шаг 8 · (Опционально) сохранил в пресет
Понравившуюся расстановку сохраняешь **именованным пресетом** и подгружаешь в другой раз — не нужно собирать заново.

### Шаг 9 · Отправил на сервер
По одной клавише (ПКМ, настраивается) расстановка уходит в **превью-объекты**, готовые к установке. Дальше:
- сбоку всплывает **умная нотификация «Установить на сервер?»** — появляется после паузы (в edit-режиме дольше, у ленты почти сразу), сама гаснет, если не реагируешь, и держится, пока на неё наведён курсор;
- есть отдельное **меню серверных объектов** — оттуда ставишь всё разом или по одному.

Объекты уходят на сервер **по очереди, по одному**, с задержками и защитой от флуда (пауза каждые 3 объекта + «отвлекающая» команда, диалог которой скрыт от игрока). По итогу — сводка «установлено X из Y».

<div align="center"><img src="assets/server-install.png" width="70%" alt="Установка на сервер: нотификация и меню"></div>

### Шаг 10 · Управляешь установленным
В меню серверных объектов — список всего, что стоит на сервере, с **дистанцией до каждого**. Удалить можно **по радиусу** (быстро, одним диалогом) или **поштучно** (бьёт строго свои объекты). А `Ctrl+Z` после установки **снимает объекты с сервера и возвращает превью** на места — полный откат.

## Режим «Лента» — все возможности

Лента строит **цельное ограждение / полицейскую ленту по контрольным точкам** — от простого отрезка до сложной трассы с провисанием и стыками. Все клавиши **переназначаются** в меню; ниже — действия по умолчанию.

**Построение**
- **Контрольные точки** — `Alt+ЛКМ` ставит точку, между точками автоматически тянется лента.
- **Сплайн или прямые** — по умолчанию плавный сплайн; `M` переключает на прямые отрезки (острые углы).
- **Прижатие к поверхности** — `Alt+Shift`: точки и лента ложатся по бордюру / стене / столбу без зазоров.
- **Курсор-режим** — `Alt`; **перетаскивание с сохранением позиции** — `Ctrl+Alt`.
- **Готово** — `ПКМ`: финализирует ленту в объекты.

**Форма и достоверный вид**
- **Прогиб** — лента провисает между опорами: `↑`/`↓` — вперёд/назад, `Shift+↑`/`↓` — вверх/вниз (вертикальное провисание, как у настоящей).
- **Высота узла** — `C` + колесо мыши.
- **Глубина утопления** — `,` / `.` : насколько лента «втоплена» в поверхность.
- **Плоская лента** — `F`: выровнять участок в плоскость.
- **Вращение участка** — колесо мыши (`2°`, `Shift 5°`, `Ctrl 0.25°`); смена плоскости вращения — `CapsLock`.

**Правка готовой ленты**
- **Выбор участка** — `←`/`→`, активный сегмент подсвечивается.
- **Правка точек** (`P`) — двигать узел камеро-относительно: `I`/`K` — вперёд/назад, `J`/`L` — влево/вправо; тащить мышью `Ctrl+Alt`; выбор цели `[` / `]`.
- **Режим стыков** (`J`) — добавлять/править швы между сегментами: выбор шва `←`/`→`, навести на ближайший `Alt`, добавить точку `Alt+ЛКМ`.
- **Сброс вращения** — `R` (а `Shift+R` — сбросить всё).
- **Пошаговый откат** — `Ctrl+Z`.

## Приятные мелочи

- **Пассивный режим** (`H`) — спрятать курсор и отдать управление игре; меню остаётся полупрозрачным оверлеем и **тускнеет, когда курсор не на нём** — не мешает обзору.
- **Удобное окно** — таскается за шапку, свободно меняет размер, **запоминает позицию и габариты**.
- **Умная нотификация установки** — всплывает не сразу, а после паузы (в edit-режиме дольше — успеть доставить объекты, у ленты почти мгновенно); **сама гаснет**, если не реагируешь, и **держится, пока курсор на ней**.
- **Защита от флуда** при установке — пауза 2-3 с каждые 3 объекта плюс «отвлекающая» команда, диалог которой **скрыт от игрока**.
- **Точное удаление с сервера** — по радиусу (быстро) или поштучно (бьёт строго свои объекты), с показом дистанции до каждого.
- **Поддержка `/objects` и `/pobjects`** — переключается в настройках под конкретный сервер.

## 📦 Установка

1. Скачайте файлы из этого репозитория (кнопка **Code → Download ZIP** или клон).
2. Разложите по папке `moonloader`:
   | Из репозитория | Куда |
   |---|---|
   | `ObjMapper.lua` | `moonloader/` |
   | содержимое `lib/` | `moonloader/lib/` |
   | содержимое `resource/` | `moonloader/resource/` |
3. Запустите игру (или `/script load`). В чат придёт приветствие — откройте меню командой **`/objhelper`**.

**Требования:** актуальный **MoonLoader**, **SA:MP 0.3.7**, **mimgui**. Остальные зависимости (EntityRender, effil, lfs, socket и др.) уже лежат в `lib/`.

> 🔄 Дальше скрипт **обновляется сам** — повторно качать вручную не нужно (см. ниже).

## 🎮 Команды и горячие клавиши

**Команда:** `/objhelper` — открыть / закрыть меню.

Основные клавиши (**все переназначаются прямо в меню**):

| Клавиша | Действие |
|---|---|
| `O` | Открыть / закрыть меню |
| `H` | Скрыть курсор / отдать управление игре |
| `1`–`9` | Выбрать объект из палитры |
| `Alt` + `1`–`9` | Активировать инструмент с этим объектом |
| `Enter` | Поставить preview (в edit-режиме) |
| `Insert` | Установить **все** preview на сервер |
| `End` | Отклонить предложение установки |
| `←` / `→` | Переключение между preview-объектами |
| `G` | Выйти из режима размещения |
| `R` | Сброс вращения |
| `Ctrl`+`Z` / `Ctrl`+`Y` | Отменить / повторить |

**Инструмент «Лента»:** `F` — плоская, `M` — без сплайна (прямые отрезки), `J` — стыки, `P` — правка точек, `C` — высота, `I/J/K/L` — сдвиг точки, `[` `]` — выбор цели, `,` `.` — глубина.

> Полный актуальный список — в блоке **«Управление»** внутри меню (страницы листаются клавишей `\`).

## 🔄 Автообновление

При запуске скрипт сверяет свою версию с этим репозиторием. Если вышла новее — в меню появляется баннер **«Обновить»** с описанием изменений, а в чат приходит уведомление. По кнопке скрипт **сам скачивает, проверяет и устанавливает** новую версию и перезагружается — ручных действий не требуется.
